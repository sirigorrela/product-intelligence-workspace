# Product Intelligence Workspace

## Overview

A Retrieval-Augmented Generation (RAG) application that enables semantic search across customer feedback, engineering reports, and product documentation using Google Gemini and ChromaDB.

## Features

- Semantic document search
- Conversational Q&A
- Gemini 2.5 Flash integration
- Chroma vector database
- LangGraph agent
- Streamlit dashboard

## Tech Stack

- Python
- Streamlit
- LangChain
- LangGraph
- Google Gemini
- ChromaDB
